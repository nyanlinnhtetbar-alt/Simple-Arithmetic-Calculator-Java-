Simple Arithmetic Calculator (Java)
A basic Java console application that performs fundamental arithmetic operations including addition, subtraction, multiplication, division with type casting, and modulo.

🚀 Features
User Input: Takes two integers from the user using the Scanner class.

Type Casting: Demonstrates converting integer division into a floating-point result to maintain precision (e.g., 9/5=1.8).

Formatted Output: Uses printf to display results, limiting the division result to one decimal place.

🛠️ Operations Included
Addition (+)

Subtraction (-)

Multiplication (*)

Division (/): Handled as a float to show decimal remainders.

Remainder/Modulo (%)

📋 How to Run
Ensure you have Java JDK installed on your machine.

Download the simple.java file.

Open your terminal or command prompt.

Compile the code:

Bash
javac simple.java
Run the program:

Bash
java simple
